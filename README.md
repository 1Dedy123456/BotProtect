Tutorial
------
Recomended for python 2.7

Using git :

    $ git clone https://github.com/farzain/BotKick
 
Python packages :

    $ pip install thrift==0.9.3
    $ pip install rsa
    $ pip install requests

Run BotKick :

    $ python botkick.py
